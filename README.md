# Anchor

Anchor is a simple link bookmarking web app. It is heavily influenced by Web 2.0 apps that are no longer with us like [Delicious](https://en.wikipedia.org/wiki/Delicious_(website)) and some that are still with us like [Pinboard](https://pinboard.in/recent/).

## Tech Stack

I'm currently building this app using libraries and frameworks I barely know how to use. So this is still pretty much a work in progress

The current stack is:
- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Prisma](https://www.prisma.io/)
- [Supabase](https://supabase.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Daisy](https://daisyui.com/)
- [NextAuth.js](https://next-auth.js.org/)
