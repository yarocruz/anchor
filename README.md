# HyperText

HypeText is a simple bookmarking web app that lets you save and organize links. It is influenced by Web 2.0 apps like [Delicious](https://en.wikipedia.org/wiki/Delicious_(website)) and [Pinboard](https://pinboard.in/recent/).

## Tech Stack

I'm currently building this app using the following technologies:
- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Prisma](https://www.prisma.io/)
- [Supabase](https://supabase.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [NextAuth.js](https://next-auth.js.org/)
